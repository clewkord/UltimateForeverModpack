# UltimateForeverModpack

Paczka Minecraft do instalacji przez moda **ServerModSync**.

## Instalacja

W grze: lista serwerow -> ikona ServerModSync -> **"Zainstaluj pelna paczke z GitHub"** -> wklej:

```
https://github.com/clewkord/UltimateForeverModpack
```

## Zawartosc

- `mods/` - 192 plikow (674M)
- `shaderpacks/` - 5 plikow (8.1M)
- `config/` - 389 plikow (1.9M)

Generowane automatycznie ze zrodlowej paczki. Repo jest lustrem paczki lokalnej.

## Uwagi

- **Physics Mod (Pro)** znajduje sie w tej paczce za zgoda autora (Haubna).
- Pliki wieksze niz 90 MB sa przechowywane w czesciach (`.part00`, `.part01`)
  i skladane automatycznie podczas instalacji przez moda ServerModSync.
